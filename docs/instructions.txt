   SEQ files (pQES magic) — 95 total in SOUND.CDB:
   - PS1 equivalent of MIDI — note events + timing + SPU commands
   - Drive the SPU hardware with precise timing
   - Sizes: up to 209KB (largest = main BGM themes)
   - Sound_Manager (0x8011d198) parses and plays these
   
   SPU sample banks (ADPCM data) — NOT in SOUND.CDB:
   Candidates:
   a) XA.MXA (85MB)    → XA-ADPCM streaming for cutscenes/BGM
   b) MDT.CDB (1.5MB)  → likely VH+VB (voice header + voice body)
                          companion files for the SEQ bank
   c) MODULE.BIN overlays → pre-load SPU RAM banks per area
